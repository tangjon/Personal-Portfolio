Goal

Migrate Drupal website to WordPress website for easier accessibility for volunteers and newer look.

Detail

Developed with basic HTML and Bootstrap. Leveraged bootstrap to developed own custom WordPress theme. Added some custom WordPress theme customizors in the admin panel. The website was developed for the 33rd Kerrisdale Scout Group. Scouts Canada is a non-profit organization that offers a world of adventure, outdoor experience and friendship to 17 million Canadian youth.

Updates
